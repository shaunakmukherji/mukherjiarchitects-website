# Cultural Exchange Center — AlUla, Saudi Arabia

## Overview
A project exploring kinetic architecture — the idea that a building need not be a static object visited once and photographed. The spaces here transform: a bright gallery becomes a dark exhibition hall, an open forum becomes an intimate performance space, a public forecourt becomes a curated event landscape. The same room accommodates fundamentally different experiences depending on the moment. Architecture designed not for the photograph, but for the return.

- **Location:** AlUla, Saudi Arabia
