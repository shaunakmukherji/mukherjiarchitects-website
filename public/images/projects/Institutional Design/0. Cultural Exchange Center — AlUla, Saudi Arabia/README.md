# Project Folder Structure

## Image Files
Place your 3 render images in this folder with the following naming convention:
- `render-01.jpg` (or .png) - Main render/preview image
- `render-02.jpg` (or .png) - Second render
- `render-03.jpg` (or .png) - Third render

**Important:** All images should be in **4:3 aspect ratio** for consistent visual appearance on the website.

## Description File
Edit `description.md` to add your project write-up. This content will be displayed when users click on the project.

## Image Specifications
- **Aspect Ratio:** 4:3 (e.g., 1600x1200px, 1920x1440px, 2400x1800px)
- **Format:** JPG or PNG
- **Recommended Size:** Under 2MB per image for fast loading
- **Naming:** Use lowercase with hyphens (render-01.jpg)
