# Five-Star Hotel, New Delhi

## Overview
A 570-key five-star hotel concept near the Lotus Temple, designed to maximise site potential, frontage, height, and premium skyline views. The proposal uses a restrained neoclassical architectural language to create a strong landmark presence, positioning the hotel as a new monument within the New Delhi skyline.

## Technical Details
- **Location:** New Delhi, India
- **Year:** 2025
- **Category:** Hospitality Design
- **Keys:** 570
- **Plot Area:** 9,129.72 sq. m. / 98,272.26 sq. ft.
- **Built-Up Area:** 34,236 sq. m. / 368,518 sq. ft.
- **Ground Coverage:** 39.9% / 3,548 sq. m.
