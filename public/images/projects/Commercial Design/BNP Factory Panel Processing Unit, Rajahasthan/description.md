# BNP Factory Panel Processing Unit, Rajasthan

## Overview
The flagship production campus for BNP Interiors was required to be architecturally extraordinary while working entirely within a structural logic already determined by engineers. Column grid, bay dimensions, and span ratios were predefined — the architecture was derived from these constraints rather than imposed over them. The result is a building where structural rhythm becomes aesthetic language: a demonstration that constraint and ambition are not opposites.

- **Location:** Rajasthan, India
